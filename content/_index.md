# Implement CICD for Java Springboot project using Jenkins and EC2 server

#### Overview

This tutorial will guide you through implementing a very basic Continuous Integration/Continuous Deployment (CI/CD) pipeline using **Jenkins** to deploy a Spring Boot project to an **EC2** instance as a Linux server. Given the scope of this lab, we will implement only the simplest elements to provide a basic hands-on learning experience. Now, let's dive into it!
